## Como inicar o projeto

- Copie a URL do repositório
- Abra o `git bash here`
- Rode o comando `git clone ` + URL
- Abra com o VS Code
- Abra a pasta `lib`
- Selecione o arquivo `main.lib`
- Depure o código
